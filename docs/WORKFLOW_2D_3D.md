# 2D/3D 工作流说明（与 2dmax.py / CT3D.py 1:1 对应）

## 一、2D 工作流（2dmax.py）

**原则：先完成四掩码分割，才能进行移动与规划。**

### 1) 图像导入 + 标定
- 打开正位图像、打开侧位图像（本地文件）。
- 设置 **正位 mm/px**、**侧位 mm/px**（标定）。
- 点击「**应用标定**」。

### 2) 四掩码保存记录
- 四个槽位：**正位-基准(Reference)**、**正位-矫正(Moving)**、**侧位-基准(Reference)**、**侧位-矫正(Moving)**。
- 列表显示每个槽位是否已保存（路径或「未保存」）。
- 可设置掩码保存目录、打开目录。

### 3) 分割（正/侧独立）
- **视图**：正位(front) / 侧位(side)。
- **角色**：基准区域 Reference / 矫正区域 Moving。
- **当前保存目标**：显示「当前保存目标：正位 - 基准(reference)」等。
- **引擎**：Classic / SAM(Box)。
- **画布操作**：在对应正位或侧位**画布上拖拽绘制矩形框 (box)**，松开后得到 box [x1,y1,x2,y2]。
- **预测**：根据当前 box 调用分割引擎（Classic 为连通域，SAM 为 box prompt），得到**候选列表**。
- **清空 / Undo / Redo / 预测** 按钮。
- **候选列表**：显示多个候选（可勾选），选中/勾选后：
  - **保存（分别）到当前部位**：只保存当前选中的一个候选到当前「视图+角色」。
  - **保存（合并）到当前部位**：将勾选的多个候选合并为一张 mask，保存到当前「视图+角色」。
- 保存后：该槽位掩码写入文件并显示在画布上（Reference 红色半透明、Moving 青色半透明），**四区域状态**更新。

### 4) 四区域状态
- 必须四个槽位都有掩码：**正位[Ref ✅ / Mov ✅] 侧位[Ref ✅ / Mov ✅]**。
- 未完成时：**移动按钮不可用**，提示「请先完成四个掩码（正Ref/正Mov/侧Ref/侧Mov）」；**A* 规划按钮不可用**，提示「请先完成四个掩码后再规划」。

### 5) 实时联动矫正（碰撞自动拦截）
- 显示**当前位姿 (mm)**：左右、上下、前后。
- 显示**目标位姿 (mm)**。
- **步长 (mm)** + **↑ ↓ ← → 前 后** 六个移动按钮。
- 每次移动前：用当前 pose 将两个视图的 Moving 掩码平移，与各自 Reference（膨胀 1px）做重叠检测；若**越界或重叠**则**禁止移动**并弹窗/状态栏提示，且在画布上绘制**红色碰撞 overlay**。
- 无四掩码时不允许移动。

### 6) 输入目标（mm）
- 六个输入：**左、右、上、下、前、后**（或等效的左右/上下/前后）。
- 点击「**将输入设为目标**」更新目标位姿。

### 7) A* 规划
- 点击「**A* 规划到目标**」前必须：
  - 四掩码已完成；
  - 当前姿态无碰撞/越界；
  - 目标姿态无碰撞/越界。
- 规划成功后：路径 overlay、**日步进**（上一天/下一天）、**自动播放**（间隔、循环）、「**应用当前预览日为当前姿态**」、「**导出 PDF 报告**」。

## 二、3D 工作流（CT3D.py）

### 1) 导入 STL
- **导入初始 STL**：可多次导入，形成**模型列表**。
- **删除所选** / **清空全部**。

### 2) 参考件（固定）
- 在列表中选择**一个**模型，点击「**设为参考**」。
- 参考件在规划时固定不动；可「**清除参考**」。

### 3) 目标调整（选择 1 个模型）
- 从列表选择**一个非参考**模型进行目标位姿调整。
- **步长 (mm)** + **X± / Y± / Z±**（或 X-左、X+右、Y-后、Y+前、Z-下、Z+上）。
- **鼠标旋转目标**：勾选后可在 3D 视图中旋转该模型目标位姿。
- **重置目标**：将目标位姿重置为当前位姿。
- 可选「隐藏非参考原件」等。

### 4) 校验目标碰撞
- 点击「**校验目标碰撞（精确）**」：检测所有模型在**目标位姿**下是否两两碰撞。
- 无碰撞则提示「目标位姿无碰撞」；有碰撞则列出碰撞对。

### 5) 规划
- **每天上限**：平移 mm、旋转 deg。
- 点击「**生成无碰撞规划（体素 A*）**」：
  - 要求已设参考、有可移动模型；
  - 先校验目标无碰撞；
  - 按顺序对每个可移动模型做体素 A* 规划，生成总天数与日步进。
- **预览：前一天 / 后一天**：按全局天数预览各模型位姿。
- 规划结果文本：参考件、可移动模型数、总天数、各模型天数区间及规划器、每日平移/旋转。

### 6) 测量（可选）
- 选择两个模型，**计算最小距离**，显示最小距离及方向分解。
